TUTORIAL · EP. 01

Building a scRNA-seq Analysis Pipeline From Scratch

A walkthrough of the core steps in single-cell RNA-seq analysis — from raw sequencing reads to clustered cell populations — written for anyone getting started with the workflow.

STEP 1

From FASTQ to Count Matrix

Sequencing produces raw reads in FASTQ format, tagged with cell barcodes and UMIs (unique molecular identifiers). Before any biology can be analyzed, these reads need to be aligned to a reference genome and quantified per cell — the output is a gene-by-cell count matrix, where each entry is the number of transcripts detected for a given gene in a given cell. This is typically done with Cell Ranger (10x Genomics' pipeline) or the open-source alternative, STARsolo. 

cellranger count --id=sample1 \
  --transcriptome=refdata-gex-GRCh38 \
  --fastqs=/path/to/fastq \
  --sample=sample1

STEP 2

Quality Control (QC)

Not every "cell" in the raw matrix is a real, healthy cell — some are empty droplets, doublets, or dying cells. Three metrics are used to filter these out: the number of genes detected per cell (nFeature_RNA), total transcript counts, and the percentage of mitochondrial reads (percent.mt) — a high mitochondrial fraction usually signals a stressed or dying cell, since damaged cells leak cytoplasmic RNA and are left with a relatively higher share of mitochondrial transcripts. 

library(Seurat)
obj <- Read10X("path/to/filtered_feature_bc_matrix")
obj <- CreateSeuratObject(obj, min.cells=3, min.features=200)
obj[["percent.mt"]] <- PercentageFeatureSet(obj, pattern="^MT-")
obj <- subset(obj, nFeature_RNA > 200 & percent.mt < 15)

STEP 3

Normalization & Clustering

Sequencing depth varies cell-to-cell, so counts need to be normalized before comparison — SCTransform handles this while also regressing out unwanted technical variation (like mitochondrial content). PCA then reduces the data to its main axes of variation, UMAP projects that into two dimensions for visualization, and FindNeighbors / FindClusters group transcriptionally similar cells into clusters — the first step toward identifying distinct cell types. 

obj <- SCTransform(obj, vars.to.regress="percent.mt")
obj <- RunPCA(obj)
obj <- RunUMAP(obj, dims=1:30)
obj <- FindNeighbors(obj, dims=1:30)
obj <- FindClusters(obj, resolution=0.5)
